DJC Compiler



Project Code

DJC is a compiler designed for Diminished Java (DJ), a programming language that is a simplified version of Java. DJC converts DJ source programs into target code for the simplified instruction set architecture DISM. It ensures that the program adheres to DJ syntax, is well-typed, and includes dynamic runtime checks to minimize errors.

Built using C, FLEX, and BISON, DJC comprises modules for lexical analysis, syntax analysis, semantic analysis, type checking, and code generation, all adhering to the language's formal grammar and type system. Additionally, it supports advanced object-oriented features such as class inheritance, method overriding, and dynamic dispatch.

RUSH



Project Code

The Rapid Unix SHell (RUSH) is a simple and efficient Unix shell designed as a command line interpreter. It creates a child process to execute each user command, waiting for the process to complete before prompting for more input, and continues this loop until the user types the built-in exit command.
RUSH searches specified directories for commands, managed via the built-in path command, and supports additional built-in commands like cd and path. It also enables output redirection with the > symbol and parallel command execution with the & operator. Errors are reported with a specific error message, and the shell robustly handles various whitespace variations in user input. RUSH is built with C.
